Construindo Apps com NocoDB - parte 01

NocoDb permite a você construir aplicações úteis sem escrever uma única linha de código! NocoDb é uma plataforma open source que transforma qualquer banco de dados em uma planilha inteligente. Conecte a um banco de dados existente, ou um novo, e transforme-o em uma planilha. Permite criar grid views, galerias de imagens, formulários a partir do banco de dados. NocoDb atualmente trabalha com MySQL, PostgreSQL, Microsoft SQL Server, SQLite, Amazon Aurora, MariaDB entre outros. Permite buscar, ordenar, filtrar colunas e linhas com facilidade. Permite trabalho compartilhado em equipe através de controle de acesso. Permite divulgar as aplicações publicamente ou protegidas por senha de acesso. Permite integração com Microsoft Teams, Slack, Discord, Whatsapp, Twilio,SMS entre outros, criando workflows complexos! Permite criar APIs com facilidade.

Instalação

sudo apt install docker.io systemctl status docker docker run -d -p 8080:8080 --name nocodb -v /path/to/nocodb:/usr/app/data/ nocodb/nocodb:latest
